# SDLC Deliverables

## Deliverable Catalog

This directory contains detailed specifications for all SDLC deliverables organized by phase.

| [Analysis](./analysis/) | Analysis | Requirements Spec, Use Case Diagram, Business Rules |
| [Design](./design/) | Design | Wireframes, UI Spec, Design System |
| [Architecture](./architecture/) | Architecture | Architecture Doc, Tech Stack, API Design |
| [Development](./development/) | Development | Code Standards, Dev Guidelines, Code Review Checklist |
| [Quality](./quality/) | Quality | Test Plan, Test Cases, Quality Metrics |
| [Deployment](./deployment/) | Deployment | Deployment Plan, CI/CD Pipeline, Runbook |
| [Project](./project/) | Project | Project Charter, Sprint Plan, Risk Register |
| [Security](./security/) | Security | Security Requirements, Threat Model, Security Checklist |

## Usage

1. **Browse deliverables**: Navigate to category folders
2. **Use templates**: Copy from `templates/` directory
3. **Follow file dimensions**: Each deliverable specifies required sections
4. **Quality gates**: All deliverables have acceptance criteria

## Quality Standards

- All deliverables must pass quality gates before phase transition
- Templates include example content (few-shot)
- Standards defined in `../standards/` apply to all deliverables
