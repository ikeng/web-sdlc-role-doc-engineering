# Workflow Phases

## Research
- Understand requirements and constraints
- Score requirement completeness
- Identify objectives and success criteria
- Gather technical context

## Ideate
- Generate multiple solutions
- Compare pros/cons
- Recommend optimal approach

## Plan
- Break solution into atomic steps
- Define file structure and logic
- Specify expected results
- Request approval

## Execute
- Implement according to plan
- Store plan in `.claude/plan/`
- Request feedback at milestones

## Optimize
- Analyze implemented code
- Identify inefficiencies
- Propose improvements
- Execute after confirmation

## Review
- Evaluate against plan
- Report issues
- Request final confirmation
