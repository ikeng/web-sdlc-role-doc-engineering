## Usage

1. Copy the template to the appropriate deliverable location
2. Replace placeholder text (e.g., `[Project Name]`) with actual values
3. Follow the file dimensions specified in the role definition
4. Complete all required sections
5. Review against quality gates

## Template Conventions

- `[Placeholder]`: Replace with actual value
- `Example:`: Shows an example value or format
- `(Optional)`: Section can be omitted if not applicable
- `(Repeat as needed)`: Section can be duplicated for additional items

## Creating New Templates

When creating a new template:
1. Define the template structure
2. Include all required sections from the deliverable definition
3. Add placeholder text with examples
4. Include quality gates checklist
5. Add approval section
